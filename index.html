About me

I am an spare time OpenWrt developer mostly working on adding device and driver support to OpenWrt and Linux. You can find me on the IRC channel (#openwrt(-devel) on irc.freenode.net) and in the forums.

I am currently working on ...

  • Finishing the BCM53XX swconfig switch driver (State: mostly working on BCM5325, needs some cleaning up)
  • Adding support for RB493G (State: Both switches usable, NAND problem not started yet) RB493G support comitted, working
  • Adding support for BCM6328 to mainline linux (State: Booting/PCIe works, flash support works, ethernet works, second core does not work is disabled in hardware)
  • Adding support for BCM6362 to mainline linux (State: none.)

I plan to tackle eventually ...

  • Add support for Cisco Linksys E2000
  • Enhance atheros switch support for swconfig (expose more options, full 4k VLAN table, ACL, ... )
  • Create a second stage bootloader for WNR2000v1 and WN2000RPT
  • Clean up and enhance swconfig
